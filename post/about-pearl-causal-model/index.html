Recently I am interested in DAG and the causal inference as to Judea Pearl's work. Here is a record of my understanding.

Concepts

Front door criterion and front door path

The first thing to clarify is so-called the "Front door criterion and front door path", following is my version of understanding.
Basically, the front door criterion requires that:

  1. Xi−>X6−>YX_i -> X_6 -> YXi​−>X6​−>Y is a front door path, and in this path X6X_6X6​ blocks XiX_iXi​ to YYY.
  2. All "back door paths" from X6X_6X6​ to YYY are blocked. Like X6X_6X6​ <- A_set_of_vars -> YYY, basically a fork like path. Here, this is indeed the case.
    And basically, if we have observed a set of variables that satiesfying front door criterion, then the causal relationship between x and y is identifiable.

Back door criterion and back door path

The other concept is back door criterion and corresponding back door path. Here, we say that (X3,X4)(X_3,X_4)(X3​,X4​) or (X5,X6)(X_5, X_6)(X5​,X6​) are the "Z" set that meets the back door criterion relative to the causal inference from XiX_iXi​ to XjX_jXj​. And we then list the conditions.

  1. (X3,X4)(X_3, X_4)(X3​,X4​) is not the decendents of XiX_iXi​, my understanding is that there is no path like Xi−>X3X_i -> X_3Xi​−>X3​. But the reverse could be true.
  2. (X3,X4)(X_3, X_4)(X3​,X4​) blocks all paths (regardless of the directions, as long as it connected) pointing to XiX_iXi​ and also pointing or pointed at XjX_jXj​.

And backdoor criterion also provides an indentification tool.
